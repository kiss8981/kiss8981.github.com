리눅스 서버에 github에 올린 파일 복사를 위한 git 설치방법

yum을 업데이트해 줍니다

```
sudo yum update -y
```

git을 설치해 줍니다


```
sudo yum install git -y
```

git이 설치되었는지 버전 확인을 합니다


```
git version
```
